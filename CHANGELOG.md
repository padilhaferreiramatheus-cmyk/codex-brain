# Changelog

All notable changes to Codex Brain will be documented in this file.

This project follows a simple release log while the CLI is early. Version numbers may move quickly until the first stable release.

## 0.1.0 - Initial Public Preview

- Added local-first Codex Brain CLI.
- Added project memory files under `.agent-brain/memory/`.
- Added Codex hook installation for session start, user prompt submit, tool use, and stop events.
- Added Markdown reports with promise-vs-reality sections.
- Added next-prompt generation for session continuity.
- Added conflict checking against local memory.
- Added local smoke tests and unit tests.
- Added `/next-prompt` skill for continuing from generated prompts.
