# Roadmap

## Fase 1 - CLI local funcional

- comandos principais;
- memoria local;
- status;
- doctor.

## Fase 2 - hooks Codex estaveis

- captura de inicio de sessao;
- captura de prompt;
- captura de ferramenta;
- captura de stop.

## Fase 3 - relatorios melhores

- promessa vs realidade mais precisa;
- conflitos baseados em memoria;
- score mais util.

## Fase 4 - integracao opcional com Graphify

Somente depois que o fluxo CLI estiver confiavel.

## Fase 5 - extensao VS Code

Somente se a CLI provar valor e houver fluxo claro para UI.

## Fase 6 - SaaS

Apenas se virar produto e sem comprometer o modo local-first.
